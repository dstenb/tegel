# example for the 3-clause BSD license
# define an 'author' record
record author {
string name;
int started;
int ended;
}
# declare an argument that is a list of authors
arg author[] authors {
cmd = "-a"; # set a short format command line name
info = "list of authors"; # set the description string
}
# declare an argument that is a string (the organization name)
arg string org {
default = "Acme Corporation"; # set the default value
cmd = "--org"; # set a long format command line name
info = "organization"; # set the description string
}
# the body starts after '%%'
%%
% for a in authors.sort("started", true)
Copyright (c) \\
% if a.started == a.ended
{{ a.started }}, \\
% else
{{ a.started.str() + "-" + a.ended.str() }}, \\
% endif
{{ a.name }}
% endfor

Compiling the file

A Python script can then be generated by calling TeGeL with the following arguments:

$ tegel -b py bsd.tgl -o bsd.py

Running the generated script

Information about the script (e.g. command line flags) can be found by using the '-h' flag.

$ python bsd.py -h
usage: bsd.py [-h] [-a AUTHORS [AUTHORS ...]] [--org ORG]
Generated by TeGeL.
optional arguments:
-h, --help show this help message and exit
-a AUTHORS [AUTHORS ...]
list of authors
--org ORG organization

For example, running the script with the following arguments

$ python bsd.py -a "Richard Stallman:1993:1993" "Bill Gates:1987:1992" --org "MicroGnu"

generates

Copyright (c) 1987-1992, Bill Gates
Copyright (c) 1993, Richard Stallman
